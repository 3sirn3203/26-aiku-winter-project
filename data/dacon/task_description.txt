[배경]
데이콘 x BDA 제2회 학습자 수료 예측 AI 경진대회 과제입니다.
BDA 9기 학습자 데이터를 기반으로 BDA 10기 학습자의 수료 여부를 예측합니다.
온라인 중심 교육 환경에서 중도 이탈 위험 학습자를 조기 식별해 학습 성취도를 높이는 것이 목적입니다.

[주제]
학습자의 수료 여부를 예측하는 AI 알고리즘 개발

[설명]
BDA 9기 학습자 설문/학습 관련 tabular 데이터를 분석하여 BDA 10기 학습자의 수료 여부를 예측하는 이진 분류 문제입니다.
타깃(label) 컬럼: completed
평가 산식: Binary F1 Score
리더보드 Public Score: 전체 테스트 데이터 중 샘플링된 50%
리더보드 Private Score: 전체 테스트 데이터 100%
1차 평가: 리더보드 Private Score 100%
2차 평가: Private Score 상위 10팀 코드 및 PPT 제출 후 코드 검증

[모델링 지침]
최종 최적화 목표는 Binary F1 Score 최대화입니다.
데이터 누수 방지, 재현 가능한 파이프라인, 안정적인 전처리/특성 엔지니어링을 우선합니다.
